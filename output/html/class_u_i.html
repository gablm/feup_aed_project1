◆ NewClass()

void UI::NewClass ( std::string  option,
Student *  student 
)
private

Adds a UC and class or changes the class if a student is already in the UC. There are various checks in place: Max allocation of 25, max allocation difference of 4 and schedule conflicts The allocation difference is not verified if the session to enter is the one with the lowest occupation. That way, the UC will return to a balanced state after a while if not already in such state.

Parameters
optionThe command inputted by the user.
studentThe pointer to the student being edited
Note
Complexity: O(n^2)
